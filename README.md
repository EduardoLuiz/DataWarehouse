DataWarehouse
=============

O projeto possui a parte  em que um certo cliente comprou um conjunto de produtos, que serão armazenados em um arquivo .arff, que será posteriormente analisados pela aplicação WEKA.

Há um treinamento realizado através do método Naive Bayes, onde se procura classificar através da série histórica de dados relações de compras para grandes conjuntos de dados.
